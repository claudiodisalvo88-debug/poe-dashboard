* aggregate-score leakage: `aggregate composition` can drift into hidden score semantics
* aggregate-formula leakage: composition language can drift into formula structure pressure
* ranking/order leakage: multiple evidence units can drift into sequence, priority or comparative placement

* schema/API/database/dashboard leakage: grouping language can drift into storage, payload, endpoint or UI grouping pressure
* implementation leakage: aggregate can drift into runtime behavior, processing path or system design language
