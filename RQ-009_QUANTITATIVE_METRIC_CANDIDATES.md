## Context

* RQ-009 tested candidate quantitative metrics for:
  * IE-002 Consumo evitabile
  * IE-003 Punte di carico
  * IE-005 Mancanza di flessibilita

## Candidate metrics

* IE-002: ACR_net — Avoidable Consumption Reduction
* IE-003: NPSR_net — Net Peak Stress Reduction
* IE-005: UFR_net — Useful Flexibility Response

## Clarification

These are measured-reduction candidate formulas, not value formulas.

Recognized value would still require:

* verified reduction;
* attribution to candidate contribution;
* systemic relevance;
* no invalidity condition;
* no material cross-IE harm.

## Postponement reason

The candidate metrics are coherent, measurable in principle, falsifiable and attribution-aware, but not safe protocol metrics yet because the following rules are missing:

1. validated baseline method for each IE;
2. validated function-preservation proof for IE-002;
3. validated critical-window and rebound-window rule for IE-003;
4. validated system-need and response-profile rule for IE-005;
5. validated attribution standard for multi-actor reductions;
6. validated cross-IE harm rejection rule.

## Final verdict

RQ-009: POSTPONED

No PD produced.
